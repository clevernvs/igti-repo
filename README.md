# User Search

Aplicação de filtragem de usuários e exibição de suas estatísticas.

### Objetivo

Implementar uma aplicação para filtrar usuários e mostrar estatísticas a partir do filtro definido.

### O que foi utilizado

- HTML
- CSS
- [Materialize]
- Javascript
